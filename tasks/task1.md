# Домашнее задание №1  
## Самостоятельная разработка дашборда в Yandex DataLens  

**Цель:** пройти весь цикл работы, подготовить данные, подключить источник, создать датасет, построить визуализации, собрать дашборд и опубликовать его.

## 1. Изучить документацию  

Ознакомиться с официальной документацией Yandex DataLens:  
https://yandex.cloud/ru/docs/datalens/  

Особое внимание уделить:
- источникам данных;  
- созданию датасетов;  
- вычисляемым полям;  
- типам чартов;  
- селекторам;  
- публикации.  

## 2. Подготовить данные в Google Таблицах  

Создать от 2 до 5 связанных таблиц. Между таблицами должны быть логические связи через ключевые поля (ID).  

**Минимальная структура:**
- таблица продаж (дата, ID клиента, ID продукта, количество, цена);  
- таблица клиентов (ID, регион, сегмент);  
- таблица продуктов (ID, категория, себестоимость).  

В основной таблице должно быть не менее 50 строк.  

## 3. Реализовать проект в DataLens  

- Подключить Google Sheets как источник данных.  
- Создать датасет, объединить таблицы и настроить типы полей.  
- Создать не менее двух вычисляемых показателей (например, выручка, прибыль).  
- Построить не менее пяти различных чартов.  
- Собрать единый дашборд и добавить селекторы (обязательно по дате и еще минимум один фильтр).  

Настроить корректную работу интерактивности.  

## 4. Публикация  

Открыть доступ к дашборду по ссылке и убедиться, что он доступен для просмотра.  

## Формат сдачи  

- ссылка на Google Таблицы;  
- ссылка на опубликованный дашборд;  
- краткое описание с пояснением логики и основных выводов.
