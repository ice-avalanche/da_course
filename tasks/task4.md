# Домашнее задание №4

## Вероятностные модели и анализ риска в Excel

**Цель:** научиться строить вероятностные модели, рассчитывать ожидаемые значения, оценивать риск и принимать решения в условиях неопределенности.

## Исходные данные

Используется [файл Excel](datasets/dataset3.xlsx) со следующими листами:

* **scenarios** – сценарная модель проекта;
* **projects** – агрегированные показатели проектов;
* **normal_model** – параметры нормального распределения;
* **simulation** – результаты имитационного моделирования.

## 1. Построение сценарной модели

**Лист:** `scenarios`

Задание:
* рассчитать выручку:
  ```
  выручка = продажи * цена
  ```
* рассчитать прибыль:
  ```
  прибыль = (цена - себестоимость) * продажи
  ```
* проверить, что сумма вероятностей = 1;
* кратко описать экономический смысл сценариев.

## 2. Расчет вероятностей событий

**Лист:** `scenarios`

Задание:
* рассчитать вероятность убытка;
* рассчитать вероятность, что прибыль выше заданного уровня (задай сам);
* рассчитать вероятность попадания прибыли в диапазон;
* использовать `СУММЕСЛИ`, `СУММЕСЛИМН`;
* сделать краткую интерпретацию.

## 3. Совместные события

**Лист:** `scenarios`

Задание:
* задать два события (например: высокая прибыль и высокий объем продаж);
* определить строки, где выполняются оба условия;
* рассчитать:
  * P(A)
  * P(B)
  * P(A ∩ B)
  * P(A ∪ B)
* сделать вывод о зависимости событий.

## 4. Математическое ожидание

**Лист:** `scenarios`

Задание:
* рассчитать математическое ожидание прибыли (1) вручную, (2) через `СУММПРОИЗВ`;
* сравнить результаты и сделать вывод.

## 5. Оценка риска

**Лист:** `scenarios`

Задание:
* рассчитать отклонения прибыли от среднего;
* рассчитать дисперсию с учетом вероятностей;
* рассчитать стандартное отклонение;
* сделать вывод об уровне риска.

## 6. Сравнение проектов

**Лист:** `projects`

В таблице уже заданы:

* ожидаемая прибыль;
* стандартное отклонение;

Задание:
* рассчитать коэффициент вариации:
  ```
  CV = σ / E
  ```
* сравнить проекты по:
  * доходности;
  * риску;
* сделать вывод о предпочтительном проекте.

## 7. Нормальное распределение

**Лист:** `normal_model`

Использовать заданные параметры (μ и σ).

Задание:
* рассчитать вероятность:
  * X < заданного значения;
  * X > заданного значения;
  * X в диапазоне;
* найти значение по заданной вероятности;
* использовать:

  * `НОРМ.РАСП`;
  * `НОРМ.ОБР`.

## 8. Вероятность убытка

**Лист:** `normal_model`

Задание:
* рассчитать вероятность убытка (X < 0);
* интерпретировать результат.

## 9. Имитационное моделирование

**Лист:** `simulation`

В таблице уже есть результаты моделирования (продажи и прибыль).

Задание:
* рассчитать:
  * среднюю прибыль;
  * стандартное отклонение;
  * минимум / максимум;
  * вероятность убытка;
* построить гистограмму распределения прибыли.

## 10. Подбор параметра

Создать отдельный расчет.

Задание:
* найти цену, при которой прибыль достигает заданного уровня;
* найти точку безубыточности;
* использовать «Подбор параметра».
