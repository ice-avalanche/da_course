# Домашнее задание №3

## Выборка, доверительные интервалы и проверка гипотез в Excel

**Цель:** научиться работать с выборочными данными, строить доверительные интервалы, формулировать и проверять статистические гипотезы, корректно применять t-тесты и интерпретировать результаты.

## Исходные данные

Используется [файл Excel](datasets/dataset2.xlsx) со следующими листами:

* **sample_data** – выборка значений (одна переменная);
* **before_after** – парные наблюдения (до/после);
* **group_A_B** – две независимые группы.

## 1. Анализ выборки

**Лист:** `sample_data`

Задание:

* определить размер выборки (СЧЁТ);
* рассчитать среднее значение (СРЗНАЧ);
* рассчитать стандартное отклонение (СТАНДОТКЛОН.В);

## 2. Формулирование гипотез

**Лист:** `sample_data`

Выбрать показатель (например, время обслуживания).

Задание:

* сформулировать нулевую гипотезу H₀;
* сформулировать альтернативную гипотезу H₁;
* определить тип гипотезы (двусторонняя / левосторонняя / правосторонняя);

## 3. Доверительный интервал

**Лист:** `sample_data`

Задание:

* рассчитать среднее значение;
* рассчитать стандартное отклонение;
* определить размер выборки;
* найти критическое значение t (СТЬЮДРАСП.ОБР.ДВУСТ);
* рассчитать границу ошибки;
* построить 95% доверительный интервал;
* cделать вывод о точности оценки. 

## 4. Одновыборочный t-тест

**Лист:** `sample_data`

Задать проверяемое значение (например, 10 минут).

Задание:

* рассчитать t-статистику;
* найти критическое значение t;
* сравнить |t| и t₍crit₎;
* cделать вывод о гипотезе. 

## 5. Интерпретация p-value

**Лист:** `sample_data`

Задание:

* определить p-value (через Excel или расчет);
* сравнить с уровнем значимости α = 0.05;
* cделать вывод о статистической значимости результата. 

## 6. Парный t-тест (до/после)

**Лист:** `before_after`

Данные являются **связанными (парными)** — каждое наблюдение «после» соответствует наблюдению «до».

Задание:

* рассчитать p-value с помощью функции

  ```
  =Т.ТЕСТ(диапазон_до; диапазон_после; 2; 1)
  ```

  (двусторонний парный тест);
* использовать уровень значимости α = 0.05;
* cделать вывод, есть ли статистически значимое изменение показателя. 

## 7. Двухвыборочный t-тест (независимые группы)

**Лист:** `group_A_B`

Данные представляют **две независимые выборки**.

Задание:

* рассчитать средние значения для каждой группы;
* выполнить t-тест:

  ```
  =Т.ТЕСТ(группа_A; группа_B; 2; 3)
  ```

  (двусторонний тест для выборок с разными дисперсиями);
* использовать α = 0.05;
* сделать вывод о различии между группами. 

## 8. Ошибки статистических решений

Теоретический вопрос.

Задание:

* объяснить, что такое ошибка I рода;
* объяснить, что такое ошибка II рода;
* указать, какая ошибка возникает, если эффект есть, но тест его не обнаружил.

## 9. Связь доверительного интервала и гипотез

**Лист:** `sample_data`

Задание:

* проверить, входит ли проверяемое значение в доверительный интервал;
* сравнить результат с t-тестом;
* сделать вывод о согласованности методов.
