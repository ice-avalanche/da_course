# Домашнее задание №2

## Первичный анализ данных и базовая статистика в Excel

**Цель:** освоить базовые методы анализа данных: первичный анализ, расчет статистик, выявление выбросов, анализ распределения и исследование взаимосвязей между показателями.

## Исходные данные

Используется [файл Excel](datasets/dataset1.xlsx) с несколькими листами:

* **sales_data** – основной датасет (для большинства заданий);
* **marketing_effect** – данные для корреляции и регрессии;
* **salary_data** – данные с явным выбросом;
* **income_data** – данные для анализа распределения.

## 1. Первичный анализ данных

**Лист:** `sales_data`

Задание:

* определить количество наблюдений (СЧЁТ, СЧЁТЗ);
* найти минимальные и максимальные значения по выручке (МИН, МАКС);
* проверить наличие пропусков;
* выявить возможные аномалии.

## 2. Расчет средних значений

**Листы:**

* `sales_data` (основной анализ);
* `salary_data` (анализ выбросов).

Задание:

* рассчитать среднее, медиану и моду для выручки (`sales_data`);
* рассчитать среднее и медиану для зарплат (`salary_data`).

## 3. Анализ разброса данных

**Лист:** `sales_data`

Задание:

* рассчитать размах;
* рассчитать дисперсию (ДИСП.В);
* рассчитать стандартное отклонение (СТАНДОТКЛОН.В);
* рассчитать коэффициент вариации.

## 4. Поиск выбросов

**Листы:**

* `sales_data`;
* `salary_data`.

Задание:

* рассчитать Q1 и Q3 (КВАРТИЛЬ.ВКЛ);
* найти IQR;
* определить границы выбросов;
* добавить столбец с классификацией («выброс» / «норма»).

## 5. Анализ распределения

**Листы:**

* `sales_data`;
* `income_data`.

Задание:

* рассчитать асимметрию (СКОС);
* рассчитать эксцесс (ЭКСЦЕСС);
* построить гистограмму.

## 6. Корреляционный анализ

**Лист:** `marketing_effect`

Задание:

* рассчитать коэффициент корреляции (КОРРЕЛ);
* построить диаграмму рассеяния;
* оценить силу и направление связи между рекламой и продажами.

## 7. Линейная регрессия

**Лист:** `marketing_effect`

Задание:

* построить линейную регрессию (линия тренда или ЛИНЕЙН);
* получить уравнение модели;
* определить коэффициент R².

## Формат сдачи

* файл Excel с расчетами (можно в том же файле с отдельными листами);
* краткое описание:

  * результаты анализа;
  * выявленные выбросы;
  * интерпретация статистик;
  * выводы по взаимосвязям.
